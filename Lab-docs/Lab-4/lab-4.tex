\section{Question 1}

\textbf{ What	is	the	relationship	between	a	guest	operating	system	and	a	host	operating	
system	in	a	system	like	VMware?	What	factors	need	to	be	considered	in	choosing	the	
host	operating	system?
}

A host operating system (OS) like VMware typically has direct communication with the hardware of the host machine itself. In virtualization, the guest OS is a virtual OS that sits on top of the actual hardware OS, the host OS. Factors to be considered when choosing an host OS include the weight of the host OS. The host OS should be lightweight to allow for the guest OS and other virtual machines to be as close to the physical layer of the device as possible, the host OS's speed; memory size, performance and security.
