\section{Question 1}

\textbf{ How	is your console like the ancient TTY subsystem in Unix as	described	in	https://www.linusakesson.net/programming/tty/ ?	
}

The ancient TTY subsystem and my console are similar in the sense that both can accept input from a user and produce an output. In both operating systems, user input texts are temporarily stored in the buffer before being executed in a program. Both operating systems provide an editing buffer and some basic editing commands like backspace and clear line. Finally, both operating systems (TTY and my console) only perform actions or execute commands through some kind of process or through the kernel.
